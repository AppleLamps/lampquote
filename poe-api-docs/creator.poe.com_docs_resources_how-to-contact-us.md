# How to contact us

Copy for LLMView as Markdown

Please join us on [Discord](https://discord.gg/TKxT6kBpgm), or email us at [developers@poe.com](mailto:developers@poe.com) with any questions or feedback.

If you are a company, or if you are an individual developer with a large amount of bot usage, we can also set up a shared Slack channel for easier collaboration. Please email us at [developers@poe.com](mailto:developers@poe.com) if you are interested in this.
